Énoncé : 
	On désire Inserer les mots d'un dictionnaire dans un arbre binaire 
	dont la structure de chaque noeud est : (caractere + Booléen)
		-Le booléen indique la fin de chaque mot.
	Plus de détails , vous trouverez l'enoncé .pdf en français.
Logiciel:
	On est obligé de programmer avec un langage Z (KHAWARIZME), conçue 
	par nos professeurs de l'esi ,pour bien apprendre les structures de 
	données .En offerant la possibilité d'écrire des algorithmes dans un
	Langage algorithmique simple,les dérouler,simuler,debuger..
	Lien: http://zegour.esi.dz/Developpement/Seriez/Nouveau.htm
	N.B: 
	Ce Logiciel permet de faire une traduction automatique du langage 
	algorithmique vers le C .(dès fois avec des erreurs qu'on doit les 
	regler.)
